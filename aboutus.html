About Us

Edunology is a website created for the students who want to learn computer science. It comes with a great variety of resoures. This project is an open source project and open source project developer may contribute to it and make it more better. It contains all basic types of resources whether it is E-books, youtube channels, applications or websites. This project is made with the sole purpose of awaring the students about the available resources. Most of the resources are free to access so that learning shouldn't stop ever.
